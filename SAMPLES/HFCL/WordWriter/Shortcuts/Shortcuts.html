Keyboard shortcuts



WordWriter supports the following keyboard shortcut keys.

Keys Operations Comments
Alt-F File menu Same as clicking on File tab
Alt-H Show Home tab shortcuts Displays letters and digits next to Home tab buttons. Press Alt with letter or digit instead of clicking button. Press Esc for shortcuts to Quick Access buttons.
Alt-V Show View tab shortcuts Displays letters and digits next to View tab buttons. Press Alt with letter or digit instead of clicking button. Press Esc for shortcuts to Quick Access buttons.
F1 Keyboard shortcuts Same as
Ctrl+N New file Same as or File menu / New
Ctrl+O Open file Same as or File menu / Open
Ctrl+S Save file Same as or File menu / Save
Ctrl-P Print file Same as or File menu / Print
Ctrl-F Find Same as Home tab
Ctrl-H Replace Same as Home tab
Ctrl-Keypad-(+) Zoom in Same as View tab
Ctrl-Keypad-(-) Zoom out Same as View tab
Ctrl-Tab Tab  
Ctrl-A Select all  
Ctrl-C Copy Same as Home tab
Ctrl-V Paste Same as Home tab
Ctrl-W Paste unformatted Same as Home tab
Ctrl-X Cut Same as Home tab
Ctrl-Z Undo Same as Home tab
Ctrl-Y Redo Same as Home tab
Ctrl-B Bold Same as Home tab
Ctrl-I Italics Same as Home tab
Ctrl-U Underline Same as Home tab
Ctrl-(+) (Ctrl-Shift-(=)) Convert to superscript Same as Home tab
Ctrl-(=) Convert to subscript Same as Home tab
Ctrl-1 Set line spacing to 1 line  
Ctrl-2 Set line spacing to 2 lines  
Ctrl-5 Set line spacing to 1.5 lines  
Ctrl-(') (apostrophe) Set acute accent for next keystroke After pressing Ctrl-(') press the letter to be accented.
Ctrl-(`) (back apostrophe) Set grave accent for next keystroke After pressing Ctrl-(`) press the letter to be accented.
Ctrl-(~) (tilde) Set tilde for next keystroke After pressing Ctrl-(~) press the letter to receive the tilde.
Ctrl-(;) (semicolon) Set umlaut for next keystroke After pressing Ctrl-(;) press the letter to receive the umlaut.
Ctrl-(^) (caret) Set circumflex for next keystroke After pressing Ctrl-(^) press the letter to receive the circumflex.
Ctrl-(,) (comma) Set cedilla for next keystroke After pressing Ctrl-(,) press the letter to receive the cedilla.
Ctrl-(") (Ctrl-Shift-(')) Activate smart quotes  
Backspace Delete previous character  
Ctrl-Backspace Delete previous word  
Insert Switch between overwrite and insert modes  
Ctrl-(←) (Left arrow) Move cursor one word to the left  
Ctrl-(→) (Right arrow) Move cursor one word to the right  
Ctrl-Home Move to the beginning of the document  
Ctrl-End Move to the end of the document  
Ctrl-PageUp Move to the top of the text currently showing in the window  
Ctrl-PageDown Move to the bottom of the text currently showing in the window  
Ctrl-Delete Delete the next word.  
Alt-X Convert a Unicode hexadecimal value in the text preceding the cursor to the corresponding character For decimal values, use Alt-xxx or Alt-0xxx.
Alt-Shift-X Convert the character in the text preceding the cursor to the corresponding Unicode hexadecimal value  
Alt-0xxx (xxx = digit sequence on number pad) Insert the character with Unicode value xxx Accepts decimal values only. For hexadecimal values, use Alt-X.
Alt-xxx (xxx = digit sequence on number pad) Insert the character with Unicode value xxx for values of 256 or greater. For values less than 256, the character is based on the current keyboard Accepts decimal values only. For hexadecimal values, use Alt-X.
Ctrl-Shift-A Convert selected text to all caps. If pressed a second time, convert to original casing There are no corresponding shortcuts for conversion to lower case or proper case.
Ctrl-Shift-L Change list style Rotates among bullet, decimal number, lowercase letter, uppercase letter, lowercase Roman numeral, uppercase Roman numeral, no bullet.
Ctrl-Shift-(>) (greater than sign) Increase font size Font size increases by 1pt in the range 4pt to 11pt, and by 2pt in the range 12pt to 28pt; changes from 28pt to 36pt to 48pt to 72pt to 80pt; increases by 10pt in the range 80pt to 1630pt; and stops at 1638pt.
Ctrl-Shift-(<) (less than sign) Decrease font size Font size decreases in the reverse order from that of Ctrl-Shift->.
