Color Array
Specifies a color for a color property

 

 

  

Syntax:

 

{ <nRed> , <nGreen> , <nBlue> }

 

where each element ranges from 0 to 255

 

 

The following color arrays have predefined names in HMG.

 

In many cases, HMG color names match those used in HTML. There are a few exceptions:

 

• ██ { 0 , 255 , 0 } is called Lime in HTML but is called GREEN, COLOR_Green, LIME, or COLOR_Lime in HMG.

 

• ██ { 0 , 128 , 0 } is called Green in HTML but is called LGREEN in HMG, even though it is darker than GREEN.

 

• HTML Brown is ██ { 165 , 42 , 42 }, but HMG BROWN is ██ { 128 , 64 , 64 }.

 

• HTML Orange is ██ { 255 , 165 , 0 }, but HMG ORANGE is ██ { 255 , 128 , 64 }.

 

• HTML Pink is ██ { 255 , 192 , 203 }, but HMG PINK is ██ { 255 , 128 , 192 }.

 

• ██ { 128 , 0 , 128 } is called Purple in HMTL and PURPLE in HMG, but HMG COLOR_Purple is ██ { 160 , 32 , 240 }.
