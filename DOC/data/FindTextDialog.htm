FindTextDialog
Opens the System Find Text Dialog and Returns Selected Find Options

 

 

 

Syntax:

 

        FINDTEXTDIALOG [ ACTION | ON ACTION <OnActionProcedure> ]

                       [ FIND <cFind> ]

                       [ NOUPDOWN ]

                       [ NOMATCHCASE ]

                       [ NOWHOLEWORD ]

                       [ CHECKDOWN <lCheckDown> ]

                       [ CHECKMATCHCASE <lCheckMatchCase> ]

                       [ CHECKWHOLEWORD <lCheckWholeWord> ]

                       [ TITLE <cTitle> ]

 

 

Methods:

  

      - FindReplaceDlg.Show

      - FindReplaceDlg.Hide

      - FindReplaceDlg.Release

 

 

 

Properties:

 

      - FindReplaceDlg.IsRelease

      - FindReplaceDlg.IsOpen

      - FindReplaceDlg.Handle

      - FindReplaceDlg.Title  [ := | --> ] cTitle

      - FindReplaceDlg.Row    [ := | --> ] nRow

      - FindReplaceDlg.Col    [ := | --> ] nCol

      - FindReplaceDlg.Width  [ := | --> ] nWidth

      - FindReplaceDlg.Height [ := | --> ] nHeight

 

 

      The following values are only valid into OnActionProcedure, else are NIL

 

      - FindReplaceDlg.RetValue   --> FRDLG_CANCEL | FRDLG_FINDNEXT

      - FindReplaceDlg.Find       --> cFindText

      - FindReplaceDlg.Down       --> lDown

      - FindReplaceDlg.MatchCase  --> lMatchCase

      - FindReplaceDlg.WholeWord  --> lWholeWord
