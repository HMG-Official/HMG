@...MONTHCALENDAR / DEFINE MONTHCALENDAR
Creates a MonthCalendar control

 

 

 

Standard Syntax (xBase Style):

 

 

      @ <row>,<col>

            MONTHCALENDAR <ControlName>

            [ OF | PARENT <ParentWindowName> ]

            [ VALUE <dValue> ]

            [ RANGEMIN <dRangeMinimum> ]

            [ RANGEMAX <dRangeMaximum> ]

            [ WIDTH <nWidth> ]

            [ HEIGHT <nHeight> ]

            [ FONT <cFontName> ]

            [ SIZE <nFontsize> ]

            [ BOLD ] [ ITALIC ] [ UNDERLINE ] [ STRIKEOUT ]

            [ FONTCOLOR <aColor> ]

            [ TRAILINGFONTCOLOR <aColor> ]

            [ BACKCOLOR <aColor> ]

            [ BORDERCOLOR <aColor> ]

            [ TITLEFONTCOLOR <aColor> ]

            [ TITLEBACKCOLOR <aColor> ]

            [ TOOLTIP <cTooltip> ]

            [ NOTODAYCIRCLE ]

Remarks:

 

Clicking on the title moves the view up one level, from month view to year view, or year view to decade view, or decade view to century view.

 

Clicking on the left and right arrows moves between items at the same view level.

 

If the “Today” item is shown at the bottom, clicking it selects today's date.

 

The default width and height of the control with the default font and size shows one month in the month view. Setting a sufficiently large width or height can show multiple months, up to a maximum of 12.
