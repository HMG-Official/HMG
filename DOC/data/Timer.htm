DEFINE TIMER
Creates a Timer Control

 

 

 

Syntax:

 

 

      DEFINE TIMER <ControlName>

            OF | PARENT <ParentWindowName>

            INTERVAL <nInterval>

            ACTION <ActionProcedureName>

 

 

Properties:

 

 

      - Parent (D)

      - Interval (D)

      - Value

      - Enabled

      - Name (R)

 

      R: Read-Only

 

 

Events:

 

 

      - Action

 

 

Methods:

 

 

      - Release
