RangeMin Property
Specifies the minimun value that can be entered in a MonthCalendar, ProgressBar, Slider, or Spinner control

 

 

  

Syntax:

 

  In a ProgressBar, Slider, or Spinner control:

RangeMin <nValue> (Control Definition)

 

<WindowName>.<ControlName>.RangeMin := nValue

 

<WindowName>.<ControlName>.RangeMin --> nValue

 

  In a MonthCalendar control:

RangeMin <dValue> (Control Definition)

 

<WindowName>.<ControlName>.RangeMin := dValue

 

<WindowName>.<ControlName>.RangeMin --> dValue

 

Notes:

 

  In a Spinner control:

 

The default value is -0x80000000 = -2147483648.

 

  In a Slider control:

 

There is no default value. Both RangeMax and RangeMin values must be set.

 

  In a ProgressBar control:

 

The default value is 0.

 

  In a MonthCalendar control:

 

The default value is no minimum, which is returned as the empty date

(d"0000-00-00"). At most one range limit can be set, either RangeMax

or RangeMin or neither, but not both. Setting one releases the other.
