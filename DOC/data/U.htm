U()
Pseudofunction for a UTF-8 character constant

 

 

 

Syntax:

 

U(nCode) --> cUtf8Char

 

Description:

 

U() is a pseudofunction which is logically equivalent to HMG_UTF8CHR() but is processed at compile time instead of at runtime.

 

For a numeric constant argument, U() returns a UTF-8 string constant and can therefore be used where a string constant is required. Two such places are the initialization expression of a STATIC variable and a CASE statement of a SWITCH block.

 

The syntax U(<nCode>) is designed to be roughly equivalent to the escape sequence notation \u<nCode> in a string constant, which Harbour does not have.

 

Examples:

 

STATIC cGamma := U(0x03B3)
 
SWITCH cChar CASE 'g' ... CASE U(0x03B3) ... END SWITCH

 

See HMG Unicode String Functions and CESU-8 and Unicode Supplemental Plane Characters in HMG.
