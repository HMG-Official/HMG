Widths Property
Specifies column widths for Browse and Grid controls. It must be specified as a numeric array containing one element for each column in the control

 

 

  

Syntax:

 

Widths <anValues>

 

Sample:

 

 

            @ 10,10 GRID Grid_1 ;

                  WIDTH 620 ;

                  HEIGHT 330 ;

                  HEADERS {'Column 1','Column 2','Column 3'} ;

                  WIDTHS {140,140,140} ;

                  ITEMS { {‘1’,’2’,’3’} , {‘4’,’5’,’6’} }
