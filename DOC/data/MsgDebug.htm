MsgDebug()
Shows a Message Box With An Debugging Information

 

 

 

Syntax:

 

 

      MsgDebug ( xData1, xData2, ... )  --> Accepts as a parameters any data type, elements or objects.

                                            Returns a string with the message displayed on the screen.

 

-    Example:

Num   := 10

aData := { "Number", 38, "aRGB", YELLOW, "Hello" }

cMsg  := MsgDebug ( TIME(), aData, {|| NIL}, Num == 3 )

 

 

 

Alternate Syntax:

 

 

      DEBUGINFO [ STOREIN cVar ];

                [ TITLE xTitle ];

                [ TYPE nTypeIconButton ];

                [ TIMEOUT nMilliseconds ];

                [ PARAMETERS ] xData1, xData2, ...  --> Accepts as a parameters any data type, elements or objects.

                                                        Returns a string with the message displayed on the screen.

 

-    Example:

LOCAL cVar

DEBUGINFO STOREIN cVar ;

          TITLE "Test 001" ;

          TYPE (MB_OK + MB_ICONEXCLAMATION) ;

          TIMEOUT 2000 ;

          PARAMETERS "Today is: ", DATE(), " - ", TIME()
