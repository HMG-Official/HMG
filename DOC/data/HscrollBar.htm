HScrollBar Property
Specifies whether a horizontal scrollbar will be included in an Edit control

 

 

  

Syntax:

 

HScrollBar <lValue>

 

Sample (Control Definition):

 

 

    DEFINE EDITBOX Edit_1

        ROW    60

        COL    230

        WIDTH  120

        HEIGHT 120

        VALUE ""

        FONTNAME "Arial"

        FONTSIZE 9

        TOOLTIP ""

        ONCHANGE Nil

        ONGOTFOCUS Nil

        ONLOSTFOCUS Nil

        FONTBOLD .F.

        FONTITALIC .F.

        FONTUNDERLINE .F.

        FONTSTRIKEOUT .F.

        HELPID Nil

        TABSTOP .T.

        VISIBLE .T.

        READONLY .F.

        HSCROLLBAR .T.

        VSCROLLBAR .T.

        BACKCOLOR Nil

        FONTCOLOR Nil

    END EDITBOX
