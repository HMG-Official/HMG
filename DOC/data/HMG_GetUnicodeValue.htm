HMG_GETUNICODEVALUE()
Converts a UTF-8 string to an array of Unicode code points

 

 

 

Syntax:

 

HMG_GETUNICODEVALUE(cUniStr) --> anCodes

 

Examples:

 

HMG_GETUNICODECHARACTER("g = γ") or HMG_GETUNICODECHARACTER(E"g = \xCE\xB3") returns {0x67, 0x20, 0x3D, 0x20, 0x03B3}, where CE B3 is the UTF-8 representation of lowercase Greek gamma, Unicode code point U+03B3.
 
HMG_GETUNICODECHARACTER("😊" +E"\x0D\x0A" + "Smile") or HMG_GETUNICODECHARACTER(E"\xF0\x9F\x98\x8A\x0D\x0ASmile") returns {0x1F60A, 0x0D, 0x0A, 0x53, 0x6D, 0x69, 0x6C, 0x65}, where F0 9F 98 8A is the UTF-8 representation of the smiley face emoticon, Unicode code point U+1F60A, and 0D 0A is a carriage return line feed pair.

 

HMG_UTF8TOESCAPE() accepts both UTF-8 and CESU-8. See HMG Unicode String Functions and CESU-8 and Unicode Supplemental Plane Characters in HMG.
