Fields Property
A character array (one element per column) specifying  fields for a Browse control

 

 

  

Syntax:

 

Fields <acValues>

 

Sample:

 

DEFINE BROWSE Browse_1

            COL 10

            ROW 10

WIDTH 610

HEIGHT 390

HEADERS { 'Code' , 'First Name' , 'Last Name' }

WIDTHS { 150 , 150 , 150 }

WORKAREA Test

FIELDS { 'Test->Code' , 'Test->First' , 'Test->Last' }

VALUE 1

END BROWSE
