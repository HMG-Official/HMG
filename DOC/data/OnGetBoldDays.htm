OnGetBoldDays Event
Occurs when a MonthCalendar control can accept a list of dates to be bolded in the control

 

 

 

Syntax:

OnGetBoldDays <bOnGetBoldDays>

 

<bOnGetBoldDays> is a code block that accepts two date arguments, <dStart> and <dEnd>, and returns an array of date values, <aBold>.

 

<dStart> and <dEnd> are the earliest and latest dates that are currently visible in the control. Each date in <aBold> must be within this range. All of the dates in <aBold> are displayed in bold in the calendar. Other dates are not bolded.

 

Example:

 

The following example displays a MonthCalendar control in which the 1st and 15th days of each month are bolded:

     . . .

     @ 10,10 MONTHCALENDAR Month_1 ;
             VALUE DATE() ;
             ON GETBOLDDAYS { | dStart, dEnd | BoldDays( dStart, dEnd ) }

     . . .

     FUNCTION BoldDays ( dStart, dEnd )

       LOCAL aBold := {}
       LOCAL dTest := dStart

       DO WHILE dTest <= dEnd
         IF DAY(dTest) == 1 .OR. DAY(dTest) == 15
           AADD(aBold, dTest)
         ENDIF
         dTest ++
       END WHILE

       RETURN aBold

 

Remarks:

 

Only days can be bolded. An OnGetBoldDays block is called only in the month view, not in the year, decade, or century view.
