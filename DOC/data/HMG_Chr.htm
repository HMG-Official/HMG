HMG_CHR()
Converts a Unicode code point to a UTF-8 or CESU-8 character

 

 

 

Syntax:

 

HMG_CHR(nCode, [lCesu8]) --> cChar

 

HMG_CHR returns a UTF-8 character, unless the optional second parameter is used:

 

HMG_CHR(<nCode>) returns a UTF-8 character.
HMG_CHR(<nCode>, .T.)returns a CESU-8 character.

 

Examples:

 

HMG_CHR(0x03B3) returns "γ" (E"\xCE\xB3"), the UTF-8 representation of lowercase Greek gamma, Unicode code point U+03B3.
 
HMG_CHR(0x1F60A) returns "😊" (E"\xF0\x9F\x98\x8A"), the UTF-8 representation of the smiley face emoticon, Unicode code point U+1F60A.
 
HMG_CHR(0x1F60A, .T.) returns "��" (E"\xED\xA0\xBD\xED\xB8\x8A"), the CESU-8 representation of the smiley face emoticon. If your browser displays this as "��" instead of "😊", then it does not support CESU-8. Most browsers do not.

 

See HMG Unicode String Functions and CESU-8 and Unicode Supplemental Plane Characters in HMG.
