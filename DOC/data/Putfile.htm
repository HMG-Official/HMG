PutFile()
Opens 'Save As' System Dialog And Returns The Saved Filename

 

 

 

Syntax:

 

 

      PutFile ( acFilter , cTitle , cIniFolder , lNoChangeDir , cDefaultFileName, @cExtFile, nFilterIndex ) --> cSavedFileName

 

 

Tips:

-    @cExtFile := NIL ---> Not set selected file extension (traditional behavior)

 

-    @cExtFile := cDefaultFileExt or "" --> Sets in cExtFile the file extension selected by the user

 

-    If <lNoChangeDir> option is omitted, default value is .F.

 

-    If passed <nFilterIndex> by reference, will return index of file filter selected by user
